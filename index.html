FlexHaus: The Comeback

Win-Back Communications Pack
Tone of Voice

Warm. Direct. Momentum.

These people feel guilty already. Shame does not sell a comeback. Every message reframes the lapse as a return — "Your spot's still here" not "You haven't been in."

Voice Pillars

Warm — "Hey, we noticed." Never "You haven't been in." A human check-in from someone who remembers you.
Direct — Clear CTA, one tap, no friction. The booking link is the hero.
Momentum — Frame the return as the start of something, not the end of a gap. "Your next class is waiting."

Win-Back Sequence

3-Touch Automated Flow

Triggered when an at-risk member is identified. All copy computed from data: the class type pre-selected is their most-booking type from the CSV.

① Touch 1: SMS — 24h after identification

Channel: SMS • Trigger: Member flagged at-risk (one-and-done OR 30d+ no visit) • Metric: Link click

SMS • From: FlexHaus

Hey {name}, your spot's still here. 💪

We saved you a place in {favourite_class} this {next_weekday}. It's the one you always took — and there's a spot open.

Book it in one tap: flex.haus/comeback

② Touch 2: Email — Day 3 if no booking

Channel: Email • Trigger: No booking 72h after SMS • Metric: Email click → booking

To: {name}  |  From: Dervla <dervla@flexhaus.com>
Subject: It's been a minute. Your class is still there.

Hey {name},

It's Dervla. I run FlexHaus — I'm usually teaching, not emailing, so if you're reading this, I wanted to say it myself: your spot's still here.

No guilt. No catch. Just an open class on {day} at {time} that has your name on it if you want it.

I know the first class back is the hardest. So we kept it simple: {favourite_class}, next {weekday}, one tap.

Book My Comeback Class

See you soon,

Dervla
